Struktur Entitas Utama
Entitas	Keterangan
1. Proyek				: Proyek konstruksi (multi user & multi perusahaan)
2. Perusahaan			: Pemilik, kontraktor, konsultan MK, QS, perencana, dll
3. User					: Orang yang login & bekerja atas nama perusahaan tertentu
4. Paket Pekerjaan		: Setiap proyek memiliki beberapa paket: struktur, arsitektur, MEP
5. Hak Akses Dinamis	: Role tergantung pada perusahaan + proyek + paket (Super Admin yang bisa mengendalikan semua Aplikasi sebagai admin Aplikasi)


Dashboard akan menampilkan:
1. Progress per paket (vs RAB bobot)
2. Status approval material, gambar, metode
3. Jumlah tenaga kerja onsite
4. Notifikasi aktif (yang belum disetujui)
5. Grafik cuaca mingguan
6. Progress K3 & Insiden
7. Kalender jadwal rapat
